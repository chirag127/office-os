## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Development

```bash
# Clone the repository
git clone https://github.com/chirag127/office-os.git
cd office-os

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Multi-Platform Deployment

Deploy to all 7 platforms at once using our Python script:

```bash
# Install dependencies
cd ops
pip install -r requirements.txt

# Configure credentials (copy and edit .env.example)
cp .env.example .env

# Deploy to all platforms
python multi_deploy.py
```

Supports: **Cloudflare Pages**, **Vercel**, **Netlify**, **Surge**, **Neocities**, **GitHub Pages**, **Render**
